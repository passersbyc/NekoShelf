# 🐾 NekoShelf (猫猫书架) 🐾

> "主人，欢迎回来！我会帮您把所有的藏书都整理得整整齐齐的喵~"

NekoShelf 是一个充满萌力的本地资源管理系统，专为整理、索引和下载各类数字资源而生。它不仅能帮你管理本地的文件，还能通过强大的插件系统从网络上把宝贝叼回来喵！

## ✨ 功能特性

- **萌力十足的 CLI**：像和猫猫聊天一样管理你的书架。
- **本地书架管理**：自动扫描、索引本地资源，支持按作者、标签、系列进行分类。
- **插件化下载系统**：
  - **Pixiv**：一键下载画师作品，支持保存为 PDF 或 CBZ 格式。
  - **Kemono**：支持从镜像站同步内容。
  - **可扩展性**：可以轻松编写自己的下载插件喵！
- **SQLite 强力驱动**：所有元数据都安全地存储在本地数据库中，查询快如闪电。
- **智能导入**：支持自动解析文件名、处理重复记录以及维护作者信息。

## 🚀 快速开始

### 1. 准备环境

确保你的电脑里已经安装了 Python 3.8 或更高版本。

```bash
# 克隆项目
git clone https://github.com/yourusername/NekoShelf.git
cd NekoShelf

# 安装依赖
pip install .
```

### 2. 启动书架

直接运行 `main.py` 就可以召唤我啦：

```bash
python main.py
```

### 3. 常用指令

在控制台中，你可以尝试以下指令：

- `help`：看看我都能做些什么。
- `library scan`：让猫猫去扫描一下你的 library 文件夹。
- `query books`：查看当前书架上都有哪些宝贝。
- `download <URL>`：把外面的资源叼回来。
- `exit`：我会乖乖去休息的喵~

## 🛠️ 项目结构

- `main.py`：项目的入口，召唤萌萌的地方。
- `core/`：我的大脑所在！
  - `cli.py`：负责和你交流的界面。
  - `database.py`：负责记住所有书本信息的笔记本。
  - `download_plugins/`：猫猫叼东西的各种技巧（插件）。
  - `config.py`：所有的习惯设置都在这里。
